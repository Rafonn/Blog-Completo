Blog com Strapi, Next.js, Netlify e Heroku
Este é um blog desenvolvido utilizando Strapi como CMS (Content Management System), Next.js como framework para o frontend, Netlify para hospedagem do site estático e Heroku para hospedar a API do Strapi.

Funcionalidades
Gerenciamento de conteúdo no painel administrativo do Strapi.
Exibição de posts no frontend, criado com Next.js, com renderização dinâmica.
Deployment contínuo utilizando Netlify (frontend) e Heroku (backend).
Suporte a edição e exclusão de posts diretamente no painel do Strapi.
Como Acessar o Blog
O blog está disponível para acesso público através do seguinte link:

https://spectacular-rabanadas-7c1b0c.netlify.app

Navegue pela página inicial para ver os posts mais recentes.
Clique em qualquer post para visualizar seu conteúdo completo.

Tecnologias Utilizadas
Strapi: CMS headless para gerenciar o conteúdo do blog.
Next.js: Framework React para o frontend, permitindo renderização estática e server-side.
Netlify: Plataforma para hospedagem do site estático.
Heroku: Plataforma para hospedagem da API do Strapi.
